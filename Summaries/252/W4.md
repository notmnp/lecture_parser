## 4. Timeâ€‘Invariant vs. Timeâ€‘Varying  

**Definition (TI):**  
If an input $x(t)$ produces output $y(t)$, then a delayed input $x(t-T)$ produces a delayed output $y(t-T)$ for *any* $T$.

- **Test** (DT):  
  1. Compute $y_1[n]$ for input $x_1[n]$.  
  2. Delay input: $x_2[n]=x_1[n-m]$.  
  3. Compute $y_2[n]$.  
  4. Check if $y_2[n] = y_1[n-m]$.  

- **Example (DT):** $y[n]=n\,x[n]$  
  - $y_1[n]=n\,x_1[n]$  
  - $y_2[n]=n\,x_1[n-m]$  
  - $y_1[n-m]=(n-m)\,x_1[n-m]$  
  â†’ $y_2[n] \neq y_1[n-m]$ â†’ **timeâ€‘varying**.

- **Example (CT):** $y(t)=x(2t)$  
  - Delay input $x_2(t)=x_1(t-T)$  
  - $y_2(t)=x_1(2t-T)$  
  - $y_1(t-T)=x_1(2(t-T))=x_1(2t-2T)$  
  â†’ Not equal â†’ **timeâ€‘varying**.

## 5. Memoryless vs. Systems With Memory  

- **Memoryless:** Output depends only on *current* input.  
  $$y(t)=f\bigl(x(t)\bigr)\quad\text{or}\quad y[n]=f\bigl(x[n]\bigr)$$

- **With Memory:** Output depends on past (or future) input.  
  - **Finite memory:** Depends on a finite past window.  
  - **Infinite memory:** Depends on all past inputs.

- **Examples**  
  - *Memoryless:* $y(t)=(t-1)x(t)$  
  - *Finite memory:* $y(t)=(t-1)x(t)+x(t-3)$ (depends on $t$ and $t-3$)  
  - *Infinite memory:*  
    $$y[n] = y[n-1] + x[n] = \sum_{k=-\infty}^{n} x[k]$$  
    (Accumulator, stores sum of all past samples)

| Category | Continuousâ€‘Time (CT) | Discreteâ€‘Time (DT) |
|----------|----------------------|--------------------|
| **Signal Nature** | Analog | Digital |
| **Time Domain** | $t \in \mathbb{R}$ | $n \in \mathbb{Z}$ |
| **Key Test** | Shift input â†’ shift output | Same, but with integer shift |
| **Memory** | Depends on past realâ€‘time interval | Depends on past sample indices |
| **Typical Example** | $y(t)=\int_0^t x(\tau)d\tau$ | $y[n]=\sum_{k=0}^n x[k]$ |
